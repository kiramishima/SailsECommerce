# Micro Tiendas
Es un proyecto creado como hobby donde ire aplicando lo que he ido aprendiendo con el paso de los días, engloba mucho conocimiento.

Desde crear la infrasctructura para desarrollo hasta la infrastructura para produccion. 

Claro todo esto es simulado en maquina local y llevado a una instancia en la nube de Microsoft Azure(por que Bizspark).
Digamos que este proyecto es un mini Uber eats.

Para cada session creare un directorio, donde `infrastructure` contiene informacion de desplegue de la infrastructura de la app.
`Backend` codigo de procesos o servicios de a app, y `Frontend` la web.
